## üì¶ Installation

### Quick Install (Requires one-time Developer Mode setup)
1. Download `ninjaDark-v1.0.zip` from the Assets below
2. Extract the ZIP file to a permanent location (like `Documents\ninjaDark`)
3. Open Chrome and navigate to `chrome://extensions/`
4. Enable **Developer mode** toggle (top-right corner)
5. Click **Load unpacked** button
6. Select the extracted folder
7. Done! Visit app.ninjarmm.com to see dark mode

**Note:** Chrome requires Developer Mode for any extension not from the Chrome Web Store. This is a one-time setup.
